[译] MailChimp 内容的图片尺寸推荐

发表于 2018-08-30 | 分类于 HTML ， EDM |
字数统计: 1,454 | 阅读时长 ≈ 5

在你上传图片到 MailChimp 之前，可以参考这些图片优化建议来确保他们的显示效果。我们的建议基于你在邮件中或登录页中使用的内容区域的类型，有所不同。

这篇文章中，你将会了解我们对于 “Image”、”Image Card”、”Image + Caption” 不同区域中的图片建议。

关于图片的一些建议

以下是对你的图片的一些基本建议：

  • 格式

    当你保存图片的时候，请使用 PNG、JPG 或者 GIF 格式。如果你需要保留透明层，你可以使用 PNG 格式。

  • 颜色模式或配置

    网上使用的颜色和那些打印中的颜色是不一样的，所以确保你在图片中使用正确的颜色，是至关重要的。网络图片应该使用 RGB 颜色配置或模式。而设计师们专门用于打印的 CMYK 模式，在网上会显示不准确，甚至有些时候会损坏你插入的图片。

  • Alt 文本

    为你的图片添加 Alt 文本是一个值得推荐的想法。Alt 文本对于可读性和一些可能无法下载你的图片的用户，是必不可少的。它也能够对你的登录页的搜索引擎优化有所帮助。

  • 适配高解析度的屏幕

    现在，很多智能手机、平板和笔记本都具备高解析度的屏幕，例如 retina 显示屏，它们能在每英寸中显示更高比率的像素。这意味着有些图片在这些显示屏中，有可能会出现模糊的状态。要保证你的图片能在所有屏幕上清晰显示，你可以使用更高分辨率的图片在图片区域，而且要控制好尺寸。

  • 文件大小

    最好是要避免文件大小和像素尺寸太大的图片。我们建议一个图片文件的大小最好不要超过 1 MB。72 DPI 用于网络使用是最好的，不过也不强制要求。我们不建议上传远远大于你的内容区域或者模板的图片。

  • 图片大小

    邮件和登录页的图片有着一些限制。对于特定的宽度建议，根据你的内容来参考以下环节。

用于邮件的图片大小

MailChimp 的邮件模板默认宽度是 600px, 所以要留意你的图片不要超过这个最大宽度。

图片大小的要求，根据你的邮件模板的列数，来决定。

列数 建议图片宽度
Full-width 模板图片 500px
2 列模板图片 264px
3 列模板图片 164px

用于登录页的图片大小

MailChimp 的登录页也有着不同的宽度，所以理想的图片大小取决于你登录页的宽度。在你上传图片到 MailChimp 之前，缩放好需要的大小，来去除不需要的像素。

如果你设置你的登录页与图片等宽，基于浏览用户的不同屏幕分辨率，效果也会有所不同。这种情况下，我们建议你使用更大的图片。不过需要注意更大的图片有可能会导致你的登录页变慢。

修改登录页宽度

跟着以下步骤来修改你的登录页宽度：

  1. 去往你的 “Campaigns” 页面；

  2. 点击你需要修改的登录页草稿；

  3. 点击 “Design“ 标签；

  4. 点击 “Page“；

  5. 在 “Page Container Style“ 部分选择相应的宽度；

    1. 点击 “Save“。

登录页的背景图片

较大的背景图在大屏幕上显示更清晰，不过我们建议图片的宽度维持在 1500px 到 2500px 之间。推荐的图片高度则取决于你在登录页中的内容篇幅，不过切记大图片会让你的页面加载变慢。
