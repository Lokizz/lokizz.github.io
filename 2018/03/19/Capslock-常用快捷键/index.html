Capslock+ 常用快捷键

发表于 2018-03-19 |

最近在网上看见了一款快捷键神器，主要是利用了Capslock键（平常主要用于切换大小写），完美解决了诸如「上下左右」键位等问题，现在记录一些常用的操作快捷键组合，关于更为详细的用法，可以参考作者的介绍。

文本编辑

  1. Capslock + E / D / S / F （上 / 下 / 左 / 右 ）
  2. Capslock + I / K / J / L （上 / 下 / 左 / 右选中文本）
  3. Capslock + W / R （向左 / 右删除文字）
  4. Capslock + A /G （光标向左 / 右跳一个单词，对英文、代码编辑很有用）
  5. Capslock + P / ; （移动光标至行首 / 行末，等同于Home / End）
  6. Capslock + U /O (选中光标至行首 / 行末文字）
  7. Capslock + Backspace （删除光标所在行所有文字）
  8. Capslock + Enter （换行，但无论光标是否在行末都能直接换行，而不会截断原句子）

这里仅为一小部分，完整列表可按下 Capslock+F1 查看软件的详细文档。

上述的快捷键，基本囊括了文本编辑器的常用动作，尤其避免了手指过大的位移，加快输入速度。


翻译

Capslock+ 内置集成了翻译功能，可选中单词或句子以后，按下 Capslock+T键。它会直接调用有道的API进行翻译。
你也可以接入自己申请的开发者API，避免公有API超出使用限制导致不能使用。


简单计算器

直接在文本框能输入一组数字+运算符，选中后再按下Capslock+F2，即可直接出现运算结果。


窗口绑定

Capslock+中，它可以让你把常用程序绑定在 Capslock+「`（反引号）或1~8」中，让你可以快速定位相关窗口。

首先通过 Capslock + Alt + (1-8 或 `)的任意一个按键来对当前激活的窗口进行绑定 (意思是将这个窗口绑定到1-8号或反引号键其中一个) ，

然后就可以随时通过 Capslock + (1-8 或 `) 来激活或最小化对应编号的窗口了。码字时需要在程序间来回切换也完全不用移动鼠标，窗口随叫随到，用起来非常爽。

当然，除了这个，有更多的需求也可以关注一下AutoHotKey。
